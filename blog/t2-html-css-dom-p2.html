HTML, CSS and DOM - Part 2

  • What is grid based design?

    • Grid based design is used to structure contents. Different parts of the content are divided into different sized grids so that viewers can easily see the flow of the webpage at a glance and to capture their attention.

  • Whats all the hype about responsive webpages?

    • There are many kinds of devices people use to browse the internet. These devices have different resolution and designers need to make sure their websites look good on screens of all sizes. This can be done by using "min-width" property in CSS which allows the width to change based on the width of a browser.

  • What is semantic structure?

    • There are HTML elements we can understand the function of just by looking at the tag. For example, when we see tags like <img> or <table>, we know exactly what those tags are used for unlike <div> or <span>

  • Any other points you found interesting and want to reflect on?

    • I always find testing my code interesting. Sometimes the pages look as I expected, and sometimes they don't and everything is all over the place. Opening the file I'm working on and refreshing the browser to see if I have fixed the issue is really helpful.
