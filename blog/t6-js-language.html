JavaScript Language

  • In the JavaScript language, describe all the use cases for each of the following:
  • Parentheses ( )
    • Execute a function. doSomething()
    • Take parameters for a function. add(1, 2)
    • Hold conditions for control flow such as if/else statements, for and while loops.
    • Group expressions. E.g. ordering mathematical operations.
  • Brackets [ ]
    • Declare an array.
    • Access an element from an array or a character from a string using the corresponding zero-based index.
    • Access a value of an object using its key in single/double quotes.
  • Braces { }
    • Decalre an object.
    • Hold statements for control flow statements.
    • Contain the code that will be executed when a function is declared.
  • Single Quotes ' '
    • Used to hold a string value.
    • Hold a key of an object when there is a space between the key (i.e. the key is not a single word).
    • Can be used interchangeably with double quotes.
  • Double Quotes " "
    • Used to hold a string value.
    • Hold a key of an object when there is a space between the key (i.e. the key is not a single word).
    • Can be used interchangeably with single quotes.
    • JSON Only: Must be used when declaring a key for an object, and storing a string value.
